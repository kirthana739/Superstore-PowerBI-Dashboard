# Superstore Power BI Dashboard

## About
Built an interactive Power BI dashboard analyzing 
9000+ row Kaggle Superstore dataset.

## What I Built
- KPI Cards (Total Sales: 725K, Total Profit: 108K)
- Sales by Region - Bar Chart
- Profit by Category - Bar Chart
- Monthly Sales Trend - Line Chart
- Region Slicer (interactive filter)

## Key Findings
- West region has highest sales (725,457)
- Technology is most profitable
- Sales peak in November-December
- Furniture has lowest profit

## Tools Used
- Microsoft Power BI Desktop
- Dataset: Kaggle Sample Superstore

## Features
- Interactive slicers to filter by Region
- KPI cards for quick summary
- Dynamic charts that update on filter
